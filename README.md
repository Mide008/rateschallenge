# RatesChallenge ⚖️

**High-Fidelity Business Rates Analysis & Evidence Generation**

RatesChallenge is a sophisticated "Design-to-Code" platform built to empower UK business owners. It simplifies the complex process of challenging business rate valuations by providing instant, data-driven comparable property evidence.

---

## 🚀 The Vision
In a landscape where commercial property taxes are often calculated with high margins of error, RatesChallenge acts as a "Digital Shield." By synthesizing thousands of data points from the 2023 Rating List, it generates professional Evidence Bundles that help businesses secure fair valuations.

## ✨ Core Features
* **Intuitive Analysis Wizard:** A seamless, friction-free flow for property data entry.
* **Real-time Comparables:** Instant search and filtering of neighboring property rates.
* **Automated Evidence Bundles:** Professional PDF generation ready for VOA submission.
* **Secure Dashboard:** User-centric history and report management.
* **Stripe Integration:** Secure, one-click access to premium deep-dive reports.

## 🛠️ Technical Stack
* **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
* **Language:** TypeScript
* **Database & Auth:** [Supabase](https://supabase.com/)
* **Styling:** Tailwind CSS (Minimalist, Heritage Elegance Aesthetic)
* **Payments:** Stripe
* **Email:** Resend / React Email
* **Deployment:** Vercel
